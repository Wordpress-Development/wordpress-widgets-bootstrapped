== Description ==

Change default wordpress widget html to Bootstrap markup.

Code intended to be used with Philip Newcomer's [Widget Output Filters](https://wordpress.org/plugins/widget-output-filters/) and will not activate without it.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
